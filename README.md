# System monitoringu parkingu
Projekt inżynierski polegający na stworzeniu systemu monitoringu parkingu wykorzystującego analizę obrazu w czasie rzeczywistym.

Aplikacja pobiera strumień wideo z kamery IP działającej w sieci 5G, analizuje zajętość miejsc parkingowych przy użyciu modelu YOLO i prezentuje wyniki w aplikacji webowej.

## Technologie
- React (JavaScript)
- Django (Python)
- PostgreSQL
- YOLO
